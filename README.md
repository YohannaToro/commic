# COMMIC

  tecnologias:
  
 - react
 - sass
 - axios
 
Descargar el proyecto y abrir a la raiz del proyecto, ejecutar la siguiente linea de comando para instalar las dependencias:

    npm install
Ejecutar la siguiente lienea de comando para inciar la aplicación:

    npm start
